# Автоматическая обработка корпоративной почты

Проект читает письма из папки, переводит их в JSON, классифицирует по правилам и раскладывает исходные файлы по папкам категорий.
## КАТЕГОРИ
0 - Incidents (Аварии, сбои)

1 - Service_Requests (Доступы, VPN, софт)

2 - Spam (Реклама, спам)

3 - HR (счета и подобное)

4 - прочее

## Как работает пайплайн

1 `app/app.py` берет папку с письмами в формате `txt`

2 `MailParser` выделяет тему, отправителя, получателя, дату, текст, ссылки и вложения.

3 Результат в `all_mails.json`

4 `classifier/main.py` загружает `all_mails.json` и таблицу весов `classifier/config.csv`.

5 Для темы и текста ищутся маркеры из таблицы.

6 Совпадения в теме получают больший вес, потому что тема обычно короче и важнее.

7 Побеждает категория с достаточным баллом и достаточным отрывом от второго места.

8 Если балл слишком маленький или отрыв неубедительный, письмо попадает в `прочее`

исходные файлы перемещаются в папки категорий

## Быстрый запуск

```bash
./run_classifier.sh mails sorted
```

После запуска в `sorted` появятся:

* `all_mails.json`
* `result.json`


## Запуск по шагам

```bash
python app/app.py mails sorted/all_mails.json
python classifier/main.py sorted/all_mails.json mails sorted sorted/result.json
```

## Визуализация результата

```bash
python result_viewer.py sorted/result.json
```

Или через bash-скрипт:

```bash
./run_viewer.sh sorted/result.json
```

Если путь не указать, откроется `classifier/result.json`.
В окне можно посмотреть количество писем по категориям, таблицу писем и детали выбранного обращения.
