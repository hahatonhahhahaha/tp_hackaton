# Автоматическая обработка корпоративной почты

Проект читает письма из папки, переводит их в JSON, классифицирует по правилам и раскладывает исходные `.txt` файлы по папкам.

Основной классификатор работает по маркерам из `classifier/config.csv`: для найденных слов и фраз считаются баллы по категориям. Если уверенного результата нет, письмо попадает в `прочее`.

## Категории

- `incidents` — аварии, сбои, критические ошибки, недоступность сервисов.
- `service_requests` — запросы на доступы, VPN, почту, ПО и корпоративные сервисы.
- `spam` — реклама, фишинг, подозрительные ссылки, просьбы ввести пароль или данные карты.
- `hardware_faults` — проблемы с техникой и оборудованием.
- `прочее` — письма, которые не подошли ни под одну категорию уверенно.

## Как работает

1. `app/app.py` читает письма из папки.
2. `MailParser` достает тему, отправителя, получателя, дату, текст, ссылки и вложения.
3. Результат сохраняется в `all_mails.json`.
4. `classifier/main.py` читает `all_mails.json` и `classifier/config.csv`.
5. Для темы и текста ищутся маркеры.
6. Совпадения в теме имеют больший вес, потому что тема обычно важнее и короче.
7. Выбирается категория с достаточным баллом и достаточным отрывом от второго места.
8. Если балл маленький или отрыв неубедительный, письмо идет в `прочее`.
9. Исходные файлы перемещаются в папки категорий.
10. Дополнительно создаются папки `subject_words/<слово>` для слов из тем, которые длиннее 3 букв и встречаются больше чем в 3 темах.

## Установка

```bash
python3 -m venv venv
source venv/bin/activate
python3 -m pip install -r requirements.txt
```

## Быстрый запуск

Скрипт перемещает письма из входной папки, поэтому для проверки лучше запускать на копии:

```bash
rm -rf sorted mails_copy
cp -r mails mails_copy
./run_classifier.sh mails_copy sorted
```

После запуска в `sorted` появятся:

- `all_mails.json`
- `result.json`
- папки категорий
- `subject_words`, если нашлись частые слова в темах

## Запуск по шагам

```bash
python3 app/app.py mails sorted/all_mails.json
python3 classifier/main.py sorted/all_mails.json mails sorted sorted/result.json
```

## Что есть в result.json

Для каждого письма сохраняются:

- имя файла
- тема и основные поля письма
- выбранная категория
- баллы по категориям
- найденные маркеры
- причина выбора категории
- статус перемещения
- частые слова из темы, если они были найдены
